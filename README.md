# LGP Reel Generator — GitHub Actions

## 每次怎么用

1. 把四张图片放进 `photos/`
2. 命名为：
   - `photo1.jpg`
   - `photo2.jpg`
   - `photo3.jpg`
   - `photo4.jpg`
3. 修改 `texts.json`
4. Commit changes
5. 打开 GitHub repo 顶部的 **Actions**
6. 等 workflow 跑完
7. 下载 artifact：`LGP_Reel_Output`

## 输出格式

- 1080×1920
- 30fps
- H.264 MP4
- Ken Burns slow zoom
- 白字 + 黑描边
- 自动安全区排版
